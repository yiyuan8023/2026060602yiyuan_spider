# 脚本包括的api接口：
### 账号
* 账号基本信息
### 交易
* 交易概况》》数据概览
### 服务
* 服务分析》》服务分析》》售后服务单量&服务核心指标
### 商品
* 商品分析》》商品明细》》spu&sku
* 商品分析》》类目分析
### 报表
* 报表分析》》我的报表》》新建报表
  * 维度:店铺
  * 汇总周期:自然天
  * 查询日期:自定义
  * 选择指标:流量指标、交易指标、其他指标
### 客户
* 关注店铺用户》》关注店铺用户概况》》数据概况
* 品牌会员》》会员概况》》数据概况

# 其他注意事项
* 任务启动命令(指定店铺)示例: python .\Start.py -p jd1_1_09_jdsz -s '华凌 WAHIN康睿麦专卖店'
* api接口函数名带有excel后缀的是返回的excel数据 df
* Start.py 中,店铺账号account和它所需执行的任务requirement,需在BiHome系统中进行配置 @浅草20250120
